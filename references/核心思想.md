# 毛选核心思想索引 · 卡牌选材与立意的权威依据

> 每张卡必须能挂到下面某一条核心思想，并引用其**真实出处与原文**。
> 这是 Phase 2 重审卡牌的唯一依据：名字是否正统、立意是否有据、画面是否切题，都回到这里对账。
> 原文均来自 `references/maoxuan/`（中文马克思主义文库）。

## A. 认识论 / 方法论根基（"灵魂"四篇）

### A1 实事求是
- **出处**：《改造我们的学习》(1941)、《反对本本主义》(1930)
- **原文**：「无实事求是之意，有哗众取宠之心。」「一切结论产生于调查情况的末尾，而不是在它的先头。」
- **释义**：从客观实际而非主观愿望、书本教条出发；这是毛泽东思想的精髓。
- **对应卡**：实事求是、调查研究/调查走访、调查报告、拒绝本本(反对本本主义)
- **画面提示**：抽象，**忌字面**。宜用"账本/算盘/实测数据 vs 空想"的对照，而非人物。

### A2 没有调查就没有发言权（调查研究）
- **出处**：《反对本本主义》(1930)
- **原文**：「没有调查，没有发言权。」「调查就是解决问题。」
- **对应卡**：调查研究、调查走访、调查报告、接触群众/群众路线
- **画面提示**：脚沾泥的走访、田间记录本、第一手数据；区别于"实事求是"的抽象。

### A3 矛盾论（辩证法）
- **出处**：《矛盾论》(1937)
- **原文**：
  - 主要矛盾：「事物的性质，主要地是由取得支配地位的矛盾的主要方面所规定的。」
  - 对立统一：「辩证法……关于对立面的统一的学说」「又联合又斗争」
  - 内外因：「外因是变化的条件，内因是变化的根据，外因通过内因而起作用。」
  - 特殊性：矛盾的特殊性 → 具体问题具体分析。
- **对应卡**：主要矛盾、次要矛盾、对立统一、矛盾转化(主次转化)、内因外因、具体问题、量变质变
- **画面提示**：可用"猎人对熊无视蜂群(抓主)""种子顶石+雨(内外因)""钥匙配锁(具体问题)"等物件/动作隐喻。

### A4 实践论（知行）
- **出处**：《实践论》(1937)
- **原文**：「真理的标准只能是社会的实践。」「主观和客观、理论和实践、知和行的具体的历史的统一。」
- **对应卡**：改造思想、拒绝本本、（与"实事求是"互为表里）
- **画面提示**：旧手册被实战磨损取代；从干到懂。

## B. 战争与战略（"打法"主轴）

### B1 论持久战 + 三阶段
- **出处**：《论持久战》(1938)
- **原文**：「持久战的三个阶段」——「第一个阶段，是敌之战略进攻、我之战略防御」「第二个阶段，是敌之战略保守、我之准备反攻（战略相持）」「第三阶段……战略的反攻战」；反对**速胜论**与**亡国论**；「兵民是胜利之本」；「战争的目的……保存自己，消灭敌人」。
- **对应卡**：持久战、战略退却/防御、战略相持、战略反攻、盲目乐观(速胜论)、盲目悲观(亡国论)、保存实力、兵民是本/兵民为本
- **画面提示**：困境/负面（速胜幻想、亡国绝望）**用物件/态势，不丑化战士**。

### B2 十六字诀 / 集中优势兵力 / 断其一指（运动战法）
- **出处**：《中国革命战争的战略问题》(1936)
- **原文**：
  - 「敌进我退，敌驻我扰，敌疲我打，敌退我追的十六字诀。」
  - 「集中优势兵力……包围迂回战术……打运动中之敌。」
  - 「对于人，**伤其十指不如断其一指**；对于敌，击溃其十个师不如歼灭其一个师。」
- **对应卡**：敌进我退/敌驻我扰/敌疲我打/敌退我追、集中兵力/集中优势、**伤其十指 → 改名「断其一指」**、各个击破/歼灭战、运动战、战略退却、保存实力
- **说明**：用户反馈"伤其十指→断其一指更直接"——**有原文支撑，采纳改名**。

### B3 抗日游击战争的战略地位 + 根据地
- **出处**：《抗日游击战争的战略问题》(1938)、《论持久战》
- **原文**：「在全体上，运动战是主要的，游击战是辅助的；在部分上，游击战是主要的，运动战是辅助的。」
- **对应卡**：游击战（**游击状态 → 合并/改名「游击战」**）、运动战、阵地战、破袭战、建立根据地、无根据地
- **说明**：用户反馈"游击状态→游击战、立意不对"——游击战是被毛上升为**战略**的正统术语；"游击状态"非术语，**改名并修立意**（强调"无后方流动作战只能辅助、必须建根据地"）。

### B4 纸老虎论 / 战略上藐视、战术上重视
- **出处**：《和美国记者安娜·刘易斯·斯特朗的谈话》(1946)、《关于目前党的政策中的几个重要问题》(1948)
- **原文**：「一切反动派都是纸老虎。」「在战略上、在全体上藐视敌人……同时在战术上、在每一个局部上重视敌人。」
- **对应卡**：战略藐视、战术重视
- **说明**：用户反馈"战略藐视怎么是一个战士能表达的"——**正确**。这是**全局/战略尺度**的判断（敌人整体是纸老虎），**画面不能是单兵**，应是"庞然大物实为纸糊/空壳"的尺度对照。

### B5 枪杆子里面出政权 / 独立自主
- **出处**：《战争和战略问题》(1938)
- **原文**：「每个共产党员都应懂得这个真理：'枪杆子里面出政权'。」「犯右倾机会主义错误的同志否认统一战线中的独立自主。」
- **对应卡**：独立自主、保存实力、丧失主动
- **说明**：用户反馈"独立自主画面完全不对，早年有大量真实例子"——独立自主的正统语境是**统一战线中不放弃自己的武装与领导权**（如皖南/反"一切经过统一战线"）。画面宜取**党史真实意象**（保有自己的枪/旗/根据地、不交出指挥权），而非"雪地独自开路"的泛化比喻。

## C. 根据地 / 生存 / 信心

### C1 工农武装割据 · 根据地
- **出处**：《中国的红色政权为什么能够存在？》(1928)、《井冈山的斗争》(1928)
- **对应卡**：建立根据地、无根据地、建立后勤、坚壁清野
- **画面提示**：依山而建的稳固割据点、后方供给线。

### C2 星星之火，可以燎原（反悲观、看趋势）
- **出处**：《星星之火，可以燎原》(1930)
- **原文**：「它是站在海岸遥望海中已经看得见桅杆尖头了的一只航船，它是立于高山之巅远看东方已见光芒四射喷薄欲出的一轮朝日，它是躁动于母腹中的快要成熟了的一个婴儿。」
- **对应卡**：星星之火、局部优势、破局边缘、拒绝悲观、道路曲折
- **画面提示**：可保留"小火/朝日/桅杆尖"意象——这是**毛本人给的视觉比喻**，正统且优美。

### C3 自力更生 / 自己动手
- **出处**：《论联合政府》(1945)、抗战大生产（南泥湾）
- **对应卡**：自力更生、自己动手、独立自主
- **画面提示**：窑洞纺线、南泥湾开荒（真实党史事件）。

## D. 统一战线与策略

### D1 统一战线（团结—斗争—孤立）
- **出处**：《目前抗日统一战线中的策略问题》(1940)
- **原文**：「发展进步势力，争取中间势力，孤立顽固势力。」对顽固派斗争须「有理、有利、有节」；「又联合又斗争」。
- **对应卡**：统一战线、政治仗、瓦解敌军
- **说明**：用户反馈"统一战线立意对，但'双手叠在一起'很奇怪"——**采纳**。改为"团结—斗争—孤立"的态势或工农兵学商**各执其业并肩**，去掉俗套叠手。

### D2 政策和策略（区别对待）
- **出处**：《论政策》(1940)
- **原文**：「利用矛盾，争取多数，反对少数，各个击破。」
- **对应卡**：区分敌我、各个击破、底线思维、统一战线

## E. 组织 / 作风 / "各种主义"批判（矛盾系统的正统出处）

### E1 古田会议 · 纠正党内各种错误思想 ★关键
- **出处**：《关于纠正党内的错误思想》(1929 古田会议决议)
- **原文（逐条点名）**：「单纯军事观点」「极端民主化」「非组织观点（少数不服从多数）」「绝对平均主义」「流寇思想（流寇主义的政治思想）」「盲动主义残余」。
- **对应卡**：流寇主义、绝对平均、盲动主义、个人英雄(单纯军事/个人主义)、拒绝山头(宗派/非组织)、民主集中(极端民主化的对立面)
- **说明**：这是矛盾系统那批"主义"卡的**最正统出处**，立意都应改挂到此处并引原文。

### E2 反对自由主义（11 种表现）
- **出处**：《反对自由主义》(1937)
- **原文**：列举十一种自由主义，「取消思想斗争，主张无原则的和平」。
- **对应卡**：折中主义、两面派、孤芳自赏(事不关己)、机会主义、怨天尤人
- **画面提示**：典出此处，可用"无原则和稀泥""当面背后两套"。

### E3 整顿三风（主观/宗派/党八股）
- **出处**：《整顿党的作风》《反对党八股》《改造我们的学习》(1941–42)
- **原文**：「反对主观主义以整顿学风，反对宗派主义以整顿党风，反对党八股以整顿文风。」两条宗旨「惩前毖后」「治病救人」。党八股第一罪状：「空话连篇，言之无物。」
- **对应卡**：整顿作风/整顿三风、主观主义、宗派主义、形式主义(党八股=空话/形式)、教条主义/经验主义(主观主义两种)、惩前毖后、精兵简政
- **说明**：教条主义、经验主义是"主观主义"的两种表现（《改造我们的学习》），有据。

### E4 群众路线（领导方法）
- **出处**：《关于领导方法的若干问题》(1943)
- **原文**：「从群众中来，到群众中去。」「将群众分散无系统的意见集中起来……又到群众中去。」
- **对应卡**：群众路线、为人民服务、接触群众、调查研究、兵民是本

## F. 价值 / 意志 / 精神

### F1 为人民服务
- **出处**：《为人民服务》(1944)
- **原文**：「为人民利益而死，就比泰山还重。」「因为我们是为人民服务的，所以……不怕别人批评指出。」（"精兵简政"亦出此文，李鼎铭提出）
- **对应卡**：为人民服务、批评自我(不怕批评)、精兵简政

### F2 毫不利己专门利人（白求恩精神）
- **出处**：《纪念白求恩》(1939)
- **原文**：「一个高尚的人，一个纯粹的人……一个脱离了低级趣味的人。」
- **对应卡**：先锋模范、破除特权、为人民服务

### F3 愚公移山（决心 / 排除万难）
- **出处**：《愚公移山》(1945)
- **原文**：「下定决心，不怕牺牲，排除万难，去争取胜利。」
- **对应卡**：排除万难、意志衰退(反面)、战略定力、克服恐水

## 用户反馈 → 已对账结论（few-shot 落实）

| 卡 | 反馈 | 依据/处理 |
|---|---|---|
| 伤其十指 | 改"断其一指"更直接 | 原文「伤其十指不如断其一指」，**改名 断其一指** |
| 游击状态 | 改名 游击战，立意不对 | 游击战是正统战略术语，**改名+重写立意**（与 str 游击战去重/差异化） |
| 战略藐视 | 一个战士表达不了 | 纸老虎论是**战略尺度**，**画面改"庞然大物实为纸壳"** |
| 独立自主 | 画面完全不对，有真实例子 | 正统语境=统一战线中不交武装/领导权，**改用党史真实意象** |
| 实事求是 | 精髓，画面体现不出 | **忌字面**，用"实测/账本 vs 空想"对照 |
| 统一战线 | 叠手很奇怪 | 改"团结—争取—孤立"态势或各业并肩，去叠手 |
| 围魏救赵 | 三十六计，非毛选 | **剔除/替换**为正统术语（如 避实击虚→毛有论述；或换"各个击破/调动敌人"） |
| 孤军深入 | 立意不对 | 非毛选倡导项；作为**警示卡**保留则须改立意（冒进/孤立无援之戒，挂"集中兵力/不打无准备之仗"反面） |
| 弹尽粮绝 | 不应出现战士 | **困境卡一律物件/态势**（空弹药箱、断粮仓） |
| 盲目乐观 | 不丑化战士 | 负面认知卡**不画战士出丑**，用态势/符号（速胜论=小胜被巨影笼罩，无脸特写） |

> 通则（写进 Phase 2 规则）：
> 1. **困境/负面/认知错误卡 → 不出现战士正面/狼狈形象**，改物件、地形、态势、符号。
> 2. **名字必须是毛选/毛泽东思想正统术语**，三十六计/孙子兵法/通用成语一律剔除或替换。
> 3. **每张卡立意必须引此索引中的真实出处**。
> 4. **抽象精髓**（实事求是、对立统一等）用对照式物件隐喻，不硬画。
> 5. **优先党史真实事件**（南泥湾、根据地、古田会议、纸老虎）而非泛化比喻。
